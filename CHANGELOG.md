# Changelog

## 1.0.0

- First standalone Meteor Quant release.
- Cross-platform Python, React, Rust, and Docker workflows.
- Deterministic Rust and Python backtesting engines.
- Kraken public-data paper trading, including sub-minute bars.
- Python strategy plugin SDK and baseline strategies.
- MarketLM, staged MarketHybrid, and TimesFM integrations.
- Shared BTC/USD price scale for model forecast overlays.
- Experimental high-turnover strategy and optimizer artifacts intentionally excluded.
